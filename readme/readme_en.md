## Project Description
This is a Pokedex project developed in React.js, using Material UI and React Icons. The project is responsive for all screens and applies computational thinking concepts for problem solving. Features include component translation, Pokemon search filter, light and dark themes, music button, and hamburger menu. The project fetches Pokemons, lists moves and abilities, and can filter by attributes. It uses the Pokemon API to retrieve detailed information.
<br>
<br>

## Features
Responsiveness: Interface adapted for all screens.

Internationalization: Component translation for different languages.

Search Filter: Search for Pokemons by name or attributes.

Light and Dark Themes: Toggle between light and dark themes.

Music Button: Background music for a more immersive experience.

Hamburger Menu: Easy and intuitive navigation.
